# OpenForm
It is an open source application to create Survey, take feedback and create quiz

# Technology Stack
Server: Express
Client: React
Database: MongoDB

# Test in local
    git clone https://github.com/TechFlexa/OpenForm.git OpenForm
    cd OpenForm
    yarn install
    yarn start:dev

# Commands

To run the application both server as well client will be started
    yarn start:dev

To run server only
    yarn start:server

To run client only
    yarn start:client
